# python-study-faker

## 1. Подготовка

1. Склонировать проект и перейти в папку с ним;
2. Инициализировать `venv`:

    ```sh
    python -m venv ./venv
    ```
   
3. Установить все зависимости:

   ```sh
   ./venv/bin/pip install -r requirements.txt
   ```

## 2. Запуск

Запустить проект в IDE или командой:

```sh
./venv/bin/python main.py
```
